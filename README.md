# Frontend Workspace - TP Frontend

Bienvenue dans le workspace des TP frontend. Ce dépôt contient les TP CSS et les TP Frameworks (React et Vue.js).

---

## Installation

Clonez le projet puis installez toutes les dépendances via npm :

```bash
git clone git@github.com:Siori9/Frontend-workspace.git
cd Frontend-workspace
./install.sh
```
> Grâce au script `install.sh`, toutes les dépendances des sous-projets seront installées automatiquement.

## Démarrage

Pour lancer l'environnement Storybook et accéder aux TP:
```
./start-storybooks.sh
```
> Cela démarrera Storybook avec tous les TP prêts à être testés et consultés dans le navigateur.
